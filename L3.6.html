Width and height of a table is defined by the width and height properties.

Example: table { width:100%; }th {height:50px; }

TABLE TEXT ALIGNMENT

The text in a table is aligned with the text-align and vertical-align properties.

Example: td { text-align:right;}
Example for vertical alignment: td { height:50px; vertical align: bottom; }

TABLE PADDING

To control the space between the border and content in a table, use the padding property on the and th elements.

Example: td { padding:15px; }

Example: table, td, th { border:1px solid green; } th { background-color:green; color:white; }
